# 📋 プリント刺繍加工業向け案件管理システム

Print & Embroidery Project Management System

ローカルネットワーク内の複数デバイス（PC・iPad・スマホ）からアクセスできる案件管理Webアプリです。

## ✨ 主な機能

- **3つのビュー**
  - 📊 **一覧ビュー**: 全案件を表で表示。ステータス・担当者・加工種別で絞り込み。納期順ソート。
  - 🎯 **カンバンビュー**: ステータスごとに列を分けて表示。ドラッグ&ドロップでステータス更新。
  - 📅 **カレンダービュー**: 納期を月カレンダーに表示。案件をクリックで詳細表示。

- **案件管理**
  - 案件の作成・閲覧・編集・削除
  - 案件名・顧客名・納期・加工種別・数量・担当者などを管理
  - 優先度・ステータス・メモ・関連資料リンクなどを保存

- **担当者管理**
  - 担当者情報の追加・編集・削除
  - 1日の稼働可能時間を設定
  - フルタイム・パート・生産管理・デザイナーなど職種を分類

- **コピペ取り込み機能**
  - LINE・メール・チャットのテキストを貼り付け
  - 案件名・顧客名・数量・納期などを自動抽出
  - 確認・修正後に登録

- **作業予定時間集計**
  - 担当者ごとの当日の作業予定時間を表示
  - 稼働可能時間を超えているか視覚的に判定

## 📊 ステータスの流れ

```
受注前 → 受注確定 → 生産待ち → 生産中 → 検品 → 納品済
```

各ステータスはカンバンビューで視覚的に管理でき、ドラッグ&ドロップで更新できます。

## 🎯 使い方

### 新規案件の登録

#### 方法1: フォームから直接入力
1. 「➕ 新規案件」ボタンをクリック
2. 案件情報を入力
3. 「保存」をクリック

#### 方法2: コピペ取り込み機能
1. 「📥 コピペ取り込み」タブを開く
2. LINE・メール等のテキストを貼り付け
3. 「🔍 テキストから抽出」をクリック
4. 自動抽出された情報を確認・修正
5. 「登録」をクリック

### 案件の編集・削除

#### 一覧ビュー
- 「✎ 編集」をクリックしてモーダルを開く
- 情報を修正して「保存」
- 「削除」で案件削除

#### カンバンビュー
- カードをドラッグ&ドロップでステータス変更
- 「✎ 編集」で詳細情報を修正

### 担当者管理

1. 「👥 担当者管理」をクリック
2. 「➕ 新規担当者」で追加、「✎ 編集」で修正
3. 「削除」でスタッフを削除

初期データとして、フルタイム2名、パート3名、生産管理1名、デザイナー1名の7人が登録されています。
名前や稼働時間は随時編集可能です。

## 📱 マルチデバイス対応

このアプリは**モバイルレスポンシブ対応**です：

- **PC**: 全機能をフルに使用可能
- **iPad**: 表・カンバン・カレンダーをタブレットサイズで表示
- **スマホ**: コンパクト表示に自動調整。横向き表示推奨

LAN内の全デバイスから同じサーバーにアクセスするため、複数人での同時編集も可能です。

## 📝 API リファレンス

### 案件API

```http
GET    /api/projects          # 全案件取得
GET    /api/projects/:id      # 単一案件取得
POST   /api/projects          # 案件作成
PUT    /api/projects/:id      # 案件更新
DELETE /api/projects/:id      # 案件削除
```

### 担当者API

```http
GET    /api/staff             # 全担当者取得
POST   /api/staff             # 担当者作成
PUT    /api/staff/:id         # 担当者更新
DELETE /api/staff/:id         # 担当者削除（ソフトデリート）
```

### 統計API

```http
GET    /api/stats/daily-workload?date=YYYY-MM-DD  # 担当者別作業時間
```
